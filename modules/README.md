# Это папка модулей.

Каждый модуль — отдельная папка внутри `modules/`, содержащая:
- `module.json` — манифест (обязательно)
- ядро `.exe` (поле `entry` в манифесте)
- доп. файлы модуля (на усмотрение автора)

При запуске ZPUI сканирует эту папку и автоматически подхватывает все модули.
См. подробности в `MODULES.md` в корне репозитория.
